# 3280. Convert Date to Binary

<Badges>
  <LevelBadge level="easy" />
</Badges>

## Solution

```typescript filename="solution.ts" copy
export function convertDateToBinary(date: string): string {
  return date
    .split('-')
    .map((v) => parseInt(v).toString(2))
    .join('-');
}
```

### Complexity

- Time: `O(1)`
- Space: `O(1)`
