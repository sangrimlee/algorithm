LeetCode의 문제 중 **Linked List** 관련 문제 해결 및 테스트 코드 작성을 위한 `class` 및 `function`입니다.

[예시 문제](https://leetcode.com/problems/add-two-numbers/)

## `ListNode`

LeetCode에서 주어지는 `class` 입니다.

```typescript showLineNumbers
class ListNode {
  val: number;
  next: ListNode | null;
  constructor(val?: number, next?: ListNode | null) {
    this.val = val === undefined ? 0 : val;
    this.next = next === undefined ? null : next;
  }
}
```

## `createLinkedList`

숫자 배열(`number[]`)을 `ListNode`로 이루어진 **Linked List**로 변환하는 함수입니다.

```typescript showLineNumbers
function createLinkedList(arr: number[]): ListNode | null {
  const startNode = new ListNode();
  let currentNode = startNode;
  arr.forEach((value) => {
    currentNode.next = new ListNode(value);
    currentNode = currentNode.next;
  });
  return startNode.next;
}
```

## `linkedListToArr`

`ListNode`로 이루어진 **Linked List**를 숫자 배열(`number[]`)로 변환하는 함수입니다.

```typescript showLineNumbers
function linkedListToArr(head: ListNode | null): number[] {
  if (head === null) {
    return [];
  }

  const arr: number[] = [];
  let currentNode: ListNode | null = head;

  while (currentNode !== null) {
    arr.push(currentNode.val);
    currentNode = currentNode.next;
  }

  return arr;
}
```
