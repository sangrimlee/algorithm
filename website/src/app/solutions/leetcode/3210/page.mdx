## Solution

```typescript filename="solution.ts" copy
export function getEncryptedString(s: string, k: number): string {
  const i = k % s.length;
  return s.substring(i) + s.substring(0, i);
}
```

### Complexity

- Time: `O(N)`
- Space: `O(1)`
