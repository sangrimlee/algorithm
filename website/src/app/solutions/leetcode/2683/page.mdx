## Solution

```typescript filename="solution.ts" copy
export function doesValidArrayExist(derived: number[]): boolean {
  return derived.reduce((prev, curr) => prev ^ curr, 0) === 0;
}
```

### Complexity

- Time: $O(n)$
- Space: $O(1)$
