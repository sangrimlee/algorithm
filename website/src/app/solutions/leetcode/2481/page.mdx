# 2481. Minimum Cuts to Divide a Circle

## Solution

```typescript filename="solution.ts" copy
export function numberOfCuts(n: number): number {
  if (n === 1) {
    return 0;
  }
  return n % 2 === 0 ? n / 2 : n;
}
```
