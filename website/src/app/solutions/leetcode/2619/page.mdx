# 2619. Array Prototype Last

<Badge.Level level="easy" />

</Badge.Root>

## Solution

```typescript filename="solution.ts" copy
declare global {
  interface Array<T> {
    last(): T | -1;
  }
}

Array.prototype.last = function () {
  return this.length === 0 ? -1 : this[this.length - 1];
};

export {};
```
